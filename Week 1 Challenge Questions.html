Question 2

Use the "lang" attribute to present a section in French and a section in Japanese.

The basic answer seems to be: Declare the main language in the html element, and then toggle in and out of other languages by wrapping their declaration inline using the span or div element.

Get the definition for the language from this list at w3schools.com: ISO Language Codes

Testing! Will this be presented in French? Oh, how sad, declaring "lang='fr'" doesn't actually translate what I write in the html file. =:^o

So, let's just paste in some French, thanks to Microsoft's translator, coding the change by wrapping "< span lang="fr" >" and "< / span >" around the phrase in the paragraph element:

Question 3

How is "meta name=description" used?

According to W3Schools, this expression is used to "define a description of your web page." Without context, I'm not sure how helpful that is.

Metatags.org further clarifies that search engines display the content of the description as a "mini-ad" in search results. Now that's important to know!
